### âœ… What Happens Automatically

1. **Token Extraction**
   - Register/Login scripts extract token from response
   - Saved to `{{authToken}}` variable automatically
   - Console logs confirmation

   - Collection-level auth adds `Authorization: Bearer {{authToken}}`
   - Every request inherits this automatically
   - No manual header configuration needed

   - Refresh endpoint updates `{{authToken}}`
   - All future requests use new token automatically
   - No manual variable updates needed
